This is a music bot.  

Please install node.js, npm, and ffmpeg for this bot to work.

Then use npm to install dependancies

Enter: npm install discord.js request-promise ytdl-core opusscript --save

Please open the settings.json file and fill in the information neeeded for the bot to work.  

You will need the token for the bot as well as the id for the channelid that you would like the commands to work in.
